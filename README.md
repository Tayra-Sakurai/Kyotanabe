# Kyotanabe

## Description
Kyotanabe is a Windows desktop personal finance application built with WinUI 3 and .NET 8.
It helps manage accounts, categories, payment methods, and transaction items, with local data persistence via Entity Framework Core and SQLite.
The solution also includes an embedding service module (`Inari`) for AI-assisted features.
